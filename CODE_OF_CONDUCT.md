# Code of Conduct

We want this project to be welcoming, respectful, and useful for everyone.

## Expected Behavior

- Be kind, patient, and constructive.
- Assume good intent while still naming problems clearly.
- Keep discussions focused on the work and its impact.
- Respect different backgrounds, experience levels, and communication styles.

## Unacceptable Behavior

Harassment, threats, personal attacks, discriminatory language, and repeated disruptive behavior are not acceptable.

## Enforcement

Maintainers may remove comments, close issues or pull requests, or block participants when needed to protect the project. If you see a problem, contact the repository owner through GitHub.
